### 3D Graph

The project employs a 3D graph visualization technique to unveil intricate relationships between different reaction parameters and the outcome. By utilizing matplotlib, scikit-learn, and interactive widgets, this approach offers an engaging and interactive platform to explore how important parameters impact the outcome categories of nanoparticles. The resulting visualization provides an intuitive overview of the multidimensional chemical space, aiding researchers in identifying trends and patterns.

### Installation

To generate the 3D graphs, run '3D.ipynb' in jupyter notebook.

Demo test: the 'simulated.csv' file has been generated using simulated data based on real experimental data.
